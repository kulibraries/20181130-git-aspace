This hands-on workshop draws on Git and GitHub lessons from Library Carpentry and the Mozilla Science Lab to introduce version control and collaboration using these common tools. Participants will be encouraged to help one another and to apply what they have learned to their own work problems.
For more information on the Library Carpentry curriculum, please see "Library Carpentry: software skills training for library professionals". For more learning materials from the Mozilla Science Lab, please see the Lab's Resources page.
